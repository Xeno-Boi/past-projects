Frozen Fantasy
April, 2025

engine: godot 4.3 stable

Description:
   A real time strategy game set in a frozen fantasy world. For more details, please read "Project Report.pdf"

To run:
   Run "exe/Frozen Fantasy.exe"
   ** Lag might occur when starting the game, the game will load in a bit.
   ** The exe file is too large for GitHub. To view exe file, please download it from google drive.
   ** Link:https://drive.google.com/drive/folders/1Sp82KUPe7B2lJdyA6CONuhYMbd2bxR5K?usp=sharing

To view code:
1. install godot 4.3 stable
2. import "code/project.godot" in godot engine
3. open "Project" in godot engine
** Lag might occur during the loading of the project, it will load in a bit.

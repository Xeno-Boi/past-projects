Elevator Simulator
March, 2024

Description:
This project is a GUI simulation of a set of working elevators.
Elevator components and functions are represented as buttons and panels.
This project only simulates the system part of an elevator.

to run code:
1. install Qt Creator using Qt Online Installer at https://doc.qt.io/qt-6/get-and-install-qt.html
2. open the project code/assignment_3.pro with qt creator
3. click "configure" (if Qt shows the configure page)
4. run the code using qt creator (click bottom left green arrow)

Restaurant Web Server

Description
   A project about setting a web server for some restaurants.
   Includes order page, statistics, restaurant edits and a login system.
   Each folder contains a README.txt containing information about them.

Folders

1. Simple Orderpage
   A simple restaurant order page.

2. Restaurant Order Server
   A webserver that includes the order page and statistics page for some restaurants.

3. Restaurant Edit Server
   A webserver that allows addition and edits of restaurant data.

4. Restaurant Database Server
   A webserver that has a restaurant order page, an account login system and uses cookies.
